# Anthropic

**Founded:** 2021 by Dario Amodei, Daniela Amodei, and others (former OpenAI researchers)
**Key Products:** Claude (model family), [Claude Code](references/entities/claude-code.md), Claude Co-work
**Key People:** [Boris Cherny](references/entities/boris-cherny.md) (Claude Code creator), [Cat Wu](references/entities/cat-wu.md) (Claude Code / Co-work product lead), Dario Amodei (CEO), Mike Krieger (CPO), Ben Mann (co-founder, co-created permission system)

## Relationship to Claude Code

- Claude Code was Boris's side project that went viral internally — "adoption chart was just vertical, 100%"
- Safety-first culture created tension: safety teams initially blocked agentic bash execution
- Decision to release publicly was driven by safety research: "By doing this we've been able to make the model much safer"
- Enterprise-focused distribution: "Our main customer base is enterprises"
- Internal productivity metric: 150% gain per engineer since Claude Code launch

## Culture Notes (from interviews)

- No job titles (Boris: "I'm a barista, or a scone baker")
- Prototype-first (no PRDs, no Figma mocks before building)
- AI-native product speed: Cat Wu describes feature timelines compressing from six months to one month, one week, or even one day.
- Privacy-obsessed: "We can't see people's data" — even for bug reports
- Safety is the primary organizational concern: Boris's personal priority shifted from "kind of important" to "the most important thing"
