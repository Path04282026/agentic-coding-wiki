# Michael Bolin

**Role:** Tech Lead for the open-source [Codex](references/entities/codex.md) repository at [OpenAI](references/entities/openai.md)
**Previous:** Distinguished Engineer at Meta; creator of Buck build system and Nuclide IDE
**Dimensions most associated with:** [01 - Agent Loop](references/dimensions/01-agent-loop.md), [03 - Security and Sandboxing](references/dimensions/03-security-and-sandboxing.md)

## Key Contributions

- Built and open-sourced the Buck build system at Facebook (dramatically faster Android builds)
- Built the Nuclide IDE (React-based Xcode replacement)
- Thesis project "Chickenfoot" — a Firefox extension for end-user web programming (precursor to modern agent tools)
- Led the Codex open-source Rust harness development
- Implemented the kernel-enforced sandbox system in Rust

## Notable Quotes

> "Strict sandboxing written manually in Rust to guarantee the model stays within system bounds."

> "We really don't care [how Google did it]. Just because it worked there didn't mean it was going to work here."
