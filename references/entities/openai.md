# OpenAI

**Founded:** 2015 by Sam Altman, [Greg Brockman](references/entities/greg-brockman.md), Elon Musk, and others
**Key Products:** GPT model family, ChatGPT, [Codex](references/entities/codex.md), DALL-E, Sora
**Key People:** [Greg Brockman](references/entities/greg-brockman.md) (co-founder, president), [Tibo Such](references/entities/tibo-such.md) (Codex engineering lead), [Michael Bolin](references/entities/michael-bolin.md) (open-source tech lead), [Alex (Codex PM)](references/entities/alex-(codex-pm).md) (PM lead)

## Relationship to Codex

- GitHub Copilot (2022) was the first product integration of code-generating models
- Internal "10x" prototype preceded public Codex — decided not to ship because it "wasn't AGI-pilled enough"
- Cloud-first approach proved premature → pivoted to CLI → Codex App
- Open-source strategy: Apache-2.0 harness for trust and community engagement
- Internal-first development: code review tool built internally, shipped after crossing utility threshold

## Culture Notes (from interviews)

- Hackathon culture at Meta (Michael Bolin's background)
- "Build for yourself first" (Greg's principle)
- "Near-term or long-term, never medium-term" planning (Alex)
- PM as "fill-in-the-gaps" role (Alex)
- Research-led culture that engineers from big tech find requires adjustment (Michael Bolin)
