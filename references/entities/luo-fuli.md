# Luo Fuli

**Role:** Large-model lead at Xiaomi
**Dimensions most associated with:** [05 - Multi-Agent Architecture](references/dimensions/05-multi-agent-architecture.md), [04 - Context Management](references/dimensions/04-context-management.md), [09 - Philosophical Divergence](references/dimensions/09-philosophical-divergence.md)

## Key Contributions

- Interprets [OpenClaw](references/entities/openclaw.md) as an epoch-making agent framework rather than only a Claude Code plus messaging UI.
- Defines agent frameworks as the middle layer between humans and models: memory, context, model routing, scheduling, skills, and workflow structure.
- Argues that agent frameworks and models should co-evolve; stronger frameworks can compensate for model limitations and unlock mid-level or smaller models.
- Describes swarm intelligence as a practical mechanism for improving open agent frameworks.
- Connects multi-agent workflows to research productivity: ideas can be tested in parallel by different sub-agents and cross-validated quickly.

## Notable Quotes

> "I actually regard OpenClaw as... an epoch-making Agent framework."

> "The middle layer between people and models... can be made very thick."

> "You can give ten ideas to different subagents... they can also cross-validate."
