# Peter Steinberger

**Role:** Creator of [OpenClaw](references/entities/openclaw.md) (formerly ClawBot)
**Previous:** Founder & CEO of PSPDFKit (PDF framework used on 1 billion+ devices)
**From:** Rural Austria
**Dimensions most associated with:** [01 - Agent Loop](references/dimensions/01-agent-loop.md), [06 - UI and Distribution](references/dimensions/06-ui-and-distribution.md), [09 - Philosophical Divergence](references/dimensions/09-philosophical-divergence.md)

## Key Contributions

- Created PSPDFKit, spent 13 years building it, sold shares, disappeared for 3 years
- Built the 1-hour WhatsApp prototype that became OpenClaw
- Coined "agentic engineering" as distinct from "vibe coding"
- Articulated the "closing the loop" principle — agents must debug and test themselves
- Pioneered self-modifying agent architecture — the agent modifies its own source code
- Drove 6,600 commits in a single month using 5–10 parallel agents
- Multi-model workflow: Codex for complex coding, Claude for general purpose
- Organized "Claude Code Anonymous" / "Agents Anonymous" meetups

## Notable Quotes

> "I actually think vibe coding is a slur. I do agentic engineering."

> "The big secret is it needs to be able to debug and test itself."

> "People talk about self-modifying software. I just built it."

> "Building this feels better than Factorio."

> "It's hard to compete against someone who's just there to have fun."

> "Software is all about how it feels, much more so than the feature set."

> "I was annoyed that it didn't exist. So I just prompted it into existence."
