# Boris Cherny

**Role:** Creator & Engineering Lead of [Claude Code](references/entities/claude-code.md) at [Anthropic](references/entities/anthropic.md)
**Previous:** Code quality lead across all of Meta (Instagram, Facebook, Messenger, WhatsApp); author of the first O'Reilly TypeScript book
**Dimensions most associated with:** [01 - Agent Loop](references/dimensions/01-agent-loop.md), [04 - Context Management](references/dimensions/04-context-management.md), [09 - Philosophical Divergence](references/dimensions/09-philosophical-divergence.md)

## Key Contributions

- Built Claude Code as a solo side project to learn the Anthropic API (Sep 2024)
- Invented the permission prompt system (with Ben Mann) when safety teams said bash execution was "unsolvable"
- Championed "don't put the model in a box" — the Bitter Lesson applied to agent design
- Led the deletion of RAG in favor of agentic search (glob + grep)
- Designed the multi-strategy compaction system and Dream memory consolidation
- Daily workflow: 100% AI-authored code, ~20 PRs/day, codes 1/3–1/2 on iPhone

## Notable Quotes

> "The model — it just wants to use tools. That's all it wants."

> "Never bet against the model."

> "All of Claude Code has just been written and rewritten and rewritten over and over."

> "Don't build for the model of today, build for the model 6 months from now."

> "At Meta, seeing a gain of 2% in productivity — that was a year of work by hundreds of people. This 100% [at Anthropic], this is completely unheard of."

> "Half my ideas are bad and you just have to try stuff."
