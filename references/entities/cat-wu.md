# Cat Wu

**Role:** Head of Product for [Claude Code](references/entities/claude-code.md) and Co-work at [Anthropic](references/entities/anthropic.md)
**Dimensions most associated with:** [06 - UI and Distribution](references/dimensions/06-ui-and-distribution.md), [09 - Philosophical Divergence](references/dimensions/09-philosophical-divergence.md), [05 - Multi-Agent Architecture](references/dimensions/05-multi-agent-architecture.md)

## Key Contributions

- Articulates Claude Code product strategy as a fast iteration loop around current model capabilities.
- Frames PM work in AI-native products around product taste: deciding what to build when code becomes cheaper to produce.
- Defines the product split between Claude Code CLI, desktop, web/mobile, and Co-work.
- Explains the assistant-to-agent transition for Claude Code: local task success, PR verification, multi-task execution, and eventually many remote Claude instances.
- Emphasizes reliable automations: get repeated workflows to near-100% success before depending on them.

## Notable Quotes

> "As code becomes much cheaper to write, the thing that becomes more valuable is deciding what to write."

> "The hard thing is figuring out for the current model. How do you elicit the maximum capability?"

> "The 2024 generation of products were chat-based and the Claude Code generation of products is action-based."
